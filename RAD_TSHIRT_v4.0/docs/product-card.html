RAD TSHIRT v4.0

Portable QA starter kit • Drop-in template for Node.js projects
v4.0
One‑folder • No external CLIs
What it is

A lightweight, self-contained toolkit that provides manifest validation, soft CI gates (tests, lint, determinism), and a small enforcement engine — ready to copy into any Node.js repo.

Key benefits
  • Drop‑in portability — copy the folder into a repo and run the gates
  • Low friction — no global installs or special infra
  • Non‑coder friendly docs and one‑page quick starts
Core components
  • Manifest validator (consistency checks)
  • Enforcement engine (gates runner)
  • Sandbox helpers to isolate runtime artifacts
Quick start
  1. Copy RAD_TSHIRT_v4.0/ into your repo root
  2. Run node RAD_TSHIRT_v4.0/tools/validate_manifest.js
  3. Run node RAD_TSHIRT_v4.0/enforcement/enforcement_engine.js
Download template
Or copy the folder and open RAD_TSHIRT_v4.0/README_LOCAL.md for localized steps.
Who it’s for — engineering teams, PMs, small orgs that want repeatable QA with minimal setup.
